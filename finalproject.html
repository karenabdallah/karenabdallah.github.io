Trivia Explosion

As a final project for my CSC 106 class, I created an educational app called Trivia Explosion. I designed this app using concepts that I have learned in this class. I used variables, arrays, functions, logical operators, conditional statements and a lot of buttons to write the code for my app.

I built Trivia Explosion to help users test their knowledge and challenge themselves by answering some trivia questions. This app is very simple to use. It begins by asking the user to press the start button to begin. Once the user hits the start button, a science and a history category show up on the screen. The user has the choice of picking which category they want to be tested on. After choosing a category the game begins. The instructions are easy, the user will have to answer 5 multiple choice questions by clicking on the answer. The score goes up 10 points for every question the user gets correct.
