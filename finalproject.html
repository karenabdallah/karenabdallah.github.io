Trivia Explosion

As a final project for my CSC 106 class, I created an educational app called "Trivia Explosion". I created this app using the most of the concepts that I have learned in this class. I used variables, arrays, functions, logical operators and conditional statements to write the code for my educational app.

I built Trivia Explosion to help users test their knowledge and challenge themselves by answering some trivia questions. This app is simple to use, its begins by asking the user to press the start button to begin. Once the user hits the start button, a science and a history category show up on the screen. The user has the choice picking which category they want to be questioned on. After choosing a category the game begins. The instructions are easy, the user will be given 30 seconds to answer 5 multiple choice questions by clicking on the answer. The score goes up 10 points for every question the user gets correct.
